# AI-Based Complaint Analysis Dashboard

## Overview
The AI-Based Complaint Analysis Dashboard is a Python-based analytics application designed to analyze and visualize customer complaints using AI-driven concepts and data analytics techniques.

The system categorizes complaints, performs sentiment analysis, detects complaint severity, generates analytical insights, and visualizes complaint distributions through an interactive Streamlit dashboard.

This project demonstrates practical applications of:
- AI concepts
- Data analytics
- Automation
- Business intelligence
- Dashboard development

---

# Features

## Complaint Categorization
Automatically categorizes complaints into:
- Network
- Billing
- Customer Service
- Technical
- Other

## Sentiment Analysis
Analyzes customer sentiment and classifies complaints as:
- Positive
- Negative
- Neutral

## Severity Detection
Detects complaint priority levels:
- High
- Medium
- Low

## Interactive Dashboard
Built using Streamlit with:
- Interactive complaint dataset view
- Analytics summary
- Bar chart visualization
- Pie chart visualization

## Automated Reporting
Exports analyzed complaint data automatically into CSV format for structured reporting and analysis.
